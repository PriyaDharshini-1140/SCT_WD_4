# SCT_WD_4 - Professional To-Do Web Application

## 📌 Project Overview

This project was developed as part of the **SkillCraft Technology Web Development Internship**.

The objective of this task was to create a **To-Do Web Application** that enables users to efficiently manage their daily tasks by adding, organizing, editing, completing, and tracking activities with date and time scheduling.

To enhance the application beyond a basic task manager, several modern UI/UX improvements and advanced task management features were implemented, including task categories, priority levels, progress tracking, drag-and-drop reordering, dark mode, local storage integration, and responsive design.

## 💡 How It Works

1. The user enters a task name.
2. A due date and time can be assigned to the task.
3. The user selects a category and priority level.
4. Clicking the "Add Task" button adds the task to the task list.
5. Tasks are automatically stored in Local Storage.
6. Users can search for tasks using the search bar.
7. Tasks can be filtered based on completion status.
8. Users can edit task details through a modal popup.
9. Completed tasks can be marked and restored when needed.
10. Dashboard statistics update automatically.
11. The progress bar reflects overall task completion.
12. Overdue tasks are visually highlighted.
13. Tasks can be reordered using drag-and-drop functionality.
14. Users can export task data to a CSV file.
15. Theme preferences and task data remain available after page refresh.
